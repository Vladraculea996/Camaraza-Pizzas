# 🍕 Camaraza Pizzas v1.1

Aplicación Web Progresiva (PWA) para la gestión integral de Camaraza Pizzas.

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Supabase
- GitHub Pages
- Service Worker (PWA)

## Características

- Inicio de sesión con roles
- Gestión de productos y categorías
- Variantes de pizzas
- Pedidos en tiempo real
- Notificaciones Push
- Dashboard administrativo
- PWA instalable
- Funcionamiento offline (en desarrollo)
